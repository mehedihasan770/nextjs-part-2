# Premium 3D Model Marketplace

## Nexus3D

Nexus3D is a **premium 3D model selling website** where users can browse and purchase high-quality 3D models based on their preferences.  
Users can explore different categories such as **robots, food items, trees**, and many more.

From the **model details page**, users can:
- Download 3D models
- Add selected models to the cart

The platform focuses on a smooth user experience with a modern, clean, and premium UI design.

---

## Route Summary

The application contains a total of **4 main routes**:

1. **Home**
   - Displays all updates and highlighted content for users.

2. **All Models**
   - Shows a complete list of available 3D models.

3. **Model Details Page**
   - Displays detailed information about a specific 3D model.

4. **Login**
   - Allows users to log in using their email.

---

## Implemented Features & Brief Explanation

### 1. Toast Notification
- Toast notifications appear during user login.
- Users can easily understand whether the login was **successful** or if there was an **error** (incorrect email or password).

### 2. Skeleton Loading
- Skeleton loaders are shown while data is loading.
- This helps users understand that content is being fetched and improves user experience.

### 3. Responsive & Unique Design
- Fully responsive across all devices (mobile, tablet, desktop).
- Clean, modern, and visually appealing UI for a premium feel.
